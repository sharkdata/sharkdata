SHARK ("Svenskt HavsARKiv" / "Swedish Ocean Archive") contains marine environmental monitoring data from the seas surrounding Sweden. SHARKweb is the main web application where it is possible to search and download data from SHARK.

This site (SHARKdata) is another way to access the same sets of data, but the target audience is other systems who wants to harvest data and use it, or publish the data in other systems, portals, etc.

More information about the SHARKdata system is available here: SHARKdata documentation.


SHARKdata is developed by the oceanographic unit of the Swedish Meterological and Hydrological Institute (SMHI).
The software is a product of the Swedish LifeWatch project funded by the Swedish Science Council.


Contact: shark@smhi.se
